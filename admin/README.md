# AICart Admin Panel

The administrative dashboard for AICart, built with React and Vite.

## Features
- Product Management (Add, Edit, Delete)
- Order Management (View status, Update status)
- User overview

## Setup

1.  Install dependencies: `npm install`
2.  Configure `.env` (see root README).
3.  Run development server: `npm run dev`

## Deployment
Deploy this folder to Vercel. Set `VITE_SERVER_URL` to your deployed backend URL.
