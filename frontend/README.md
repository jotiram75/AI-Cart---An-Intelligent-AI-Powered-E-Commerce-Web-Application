# AICart Frontend

The customer-facing application for AICart, built with React and Vite.

## Features
- Product Browsing & Search
- Shopping Cart
- User Authentication
- Order History
- Payment Integration

## Setup

1.  Install dependencies: `npm install`
2.  Configure `.env` (see root README).
3.  Run development server: `npm run dev`

## Deployment
Deploy this folder to Vercel.
Set `VITE_SERVER_URL` to your deployed backend URL.
Set `VITE_ADMIN_URL` to your deployed admin panel URL.
